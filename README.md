# Kiwoom News Trader MVP

PyQt5 기반의 키움 OpenAPI 뉴스 트레이딩 도구입니다.

## 구성

- 진입점: `main.py`
- UI: `app/ui/`
- 매니저: `app/managers/`
- 서비스: `app/services/`
- 데이터 저장: `data/`
- 로그: `logs/`

## 주요 기능

- 키움 계좌 및 조건식 연동
- 뉴스 점수 분석
- 텔레그램 뉴스/매매 메시지 발송
- 뉴스 테마 및 시기 테마 표시
- 프로그램 재시작 버튼 지원
